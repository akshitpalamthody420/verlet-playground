What it does

Particles, constraints, and real-time interaction

Verlet Playground is an interactive 2D physics simulation built in C++ with SFML. The simulation uses particles as physics points and connects them with distance constraints. By combining those constraints, the project can create ropes, cloth sheets, and soft-body structures.

The user can drag particles, pin points in place, cut constraints, toggle wind, toggle gravity, and spawn different physics structures during runtime.

Core systems

Features

Verlet Motion

Particles store current and previous positions instead of explicit velocity, producing stable motion that works well for constraint-based simulations.

Distance Constraints

Particles are linked together by constraints that preserve distance, making ropes, cloth grids, and soft-body forms possible.

Interactive Editing

Mouse input lets users drag, pin, unpin, and cut particles or constraints while the simulation is running.

Multiple Structures

The project can spawn ropes, cloth sheets, and soft-body blobs instead of only simulating one chain of connected particles.

Wind and Gravity Toggles

Environmental forces can be enabled or disabled during runtime to test how different structures react.

Tearing and Cutting

Constraints can break under stress or be cut manually, making the simulation more dynamic than a fixed particle system.

How to use it

Controls

Input Action
Left Mouse Drag nearest particle
Right Mouse Pin or unpin nearest particle
Middle Mouse Cut constraints near the cursor
Space Spawn a pinned rope at the cursor
C Spawn a cloth sheet near the cursor
B Spawn a soft-body blob at the cursor
W Toggle wind
T Toggle tearing
G Toggle gravity
R Reset demo scene
Backspace / Delete Clear the world
Escape Quit

Run locally

Setup

Requirements

  • C++17 or newer
  • CMake
  • SFML 2.5 or newer

Build

mkdir build
cd build
cmake ..
cmake --build .
./VerletPlayground

macOS with Homebrew

brew install cmake sfml@2

mkdir build
cd build
cmake -DCMAKE_PREFIX_PATH="$(brew --prefix sfml@2)" ..
cmake --build .
./VerletPlayground
